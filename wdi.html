White Paper – Extended

Il WDI – WizTrail Difficulty Index è un indice proprietario che misura la difficoltà oggettiva di un percorso Trail/Ultra analizzando parametri reali estratti direttamente dal GPX. Non considera elementi soggettivi (atleta, tempo finale, condizioni meteo), ma esclusivamente la struttura fisica del percorso.

1) Formula del WDI

Il WDI è definito come:

WDI = w₁·FD + w₂·FVERT + w₃·FTECH + w₄·FPROFILE + w₅·FRIP

Tutti i fattori hanno valore base 1, e crescono proporzionalmente al loro peso. I pesi (w₁…w₅) dipendono dal modello (A–E) selezionato in base alla distanza.

2) I Cinque Fattori

FD — Distance Factor

Modella il ruolo della distanza nella difficoltà, usando 5 funzioni specifiche per i modelli A–E.

FD_A(D) = 0.80 + 0.30·sqrt(D/20)
FD_B(D) = 0.80 + 0.45·ln(1 + D/25)
FD_C(D) = 1.00 + 0.02·(D - 50)
FD_D(D) = 2.00 + 0.015·(D - 90)
FD_E(D) = 1.40·sqrt(D/50)

FVERT — Verticalità

Smean = Dplus / D
top95 = percentile(slopesPos, 95)
Svar   = variance(slopesPos)
FVERT = 1 + 0.015·Smean + 0.025·top95 + 0.04·Svar

FTECH — Tecnicità

steepDown = #(slope < -0.20) / total
roughness = variance(slopes)
FTECH = 1 + 0.6·steepDown + 0.4·roughness

FPROFILE — Profilo Globale

x = array(i / (n - 1))
FPROFILE = 1 + 0.25·corr(slopes, x)

FRIP — Ripidità Massima per km

smax = max( Δelev(1km)/1000 )
FRIP = 1 + 0.4·smax
