🌐 WizTrail
Sito Web Statico – HTML \& CSS
WizTrail è un sito web statico realizzato in HTML e CSS, pensato per offrire un’interfaccia leggera, veloce e facilmente navigabile.
Il progetto è pubblicato tramite GitHub Pages, rendendolo accessibile direttamente dal browser senza installazioni o download.
