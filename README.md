🌐 WizTrail
Sito Web Statico – HTML & CSS
WizTrail è un sito web statico realizzato in HTML e CSS, pensato per offrire un’interfaccia leggera, veloce e facilmente navigabile.
Il progetto è pubblicato tramite GitHub Pages, rendendolo accessibile direttamente dal browser senza installazioni o download.

🚀 Demo online
🔗 https://ciao-madesign.github.io/WizTrail/


📂 Struttura del progetto
La struttura del progetto segue un’organizzazione semplice e chiara:
WizTrail/
├── index.html        # Pagina principale del sito
├── style.css         # Foglio di stile principale
├── img/              # Cartella immagini (se presente)

└── assets/           # Risorse aggiuntive (opzionale)
